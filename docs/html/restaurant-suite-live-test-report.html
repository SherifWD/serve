Closed Defects

ID Original Failure Fix Applied Retest Result
DEF-01 Cross-branch cashier could post payments on another branch’s order. Added branch/restaurant access guards in OrderMobileController payment flow. Closed Foreign-branch payment now returns 403.
DEF-02 Ingredients endpoint was public. Moved ingredients and related stock routes behind auth:sanctum. Closed Anonymous access now returns 401.
DEF-03 Cross-branch cashier could download another branch’s receipt. Added order access guard to receipt generation. Closed Foreign-branch receipt request now returns 403.
DEF-04 Employee performance endpoint was public. Moved performance route behind auth:sanctum. Closed Anonymous access now returns 401.
DEF-05 Mobile order detail endpoint tried to resolve a table instead of an order. Rewired OrderMobileController::show to return a scoped order payload. Closed Valid order ids now return 200 with order detail.
DEF-06 Invalid refund/change action was silently accepted. Added explicit validation to TableMobileController::refundOrChangeItem. Closed Invalid actions now return 422.
